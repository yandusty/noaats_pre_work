# noaats_pre_work
10가지 주제 중 한 가지를 골라  
**AI툴과 함께 제 입맛대로 문제 정의와 해결하는 과정과 결과를 기록**하는 레포.

# 실행 방법
- ```git clone https://github.com/yandusty/noaats_pre_work/``` 으로 레포를 다운받는다.
- 해당 폴더로 디렉토리를 변경한다. (app.py가 존재하는 디렉토리까지)
- python 3.10 이상 버전으로 설치한다.
- ```pip install -r requirements.txt``` 로 필요한 패키지들을 설치해준다.
- ```streamlit run app.py```를 입력해 프로토타입을 실행한다.
